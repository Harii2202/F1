1) Factorial of a Number

Q1: How do you take a number input from the user?

Q2: How do you initialize the result variable for factorial calculation?

Q3: How do you use a loop to multiply numbers from 1 to the input number?

Q4: How do you handle the case when the input number is 0?

Q5: How do you display the final factorial result?

Pseudocode:

START
    INPUT number
    IF number == 0
        PRINT 1
    ELSE
        SET factorial = 1
        FOR i from 1 to number
            SET factorial = factorial * i
        ENDFOR
        PRINT factorial
    ENDIF
END
